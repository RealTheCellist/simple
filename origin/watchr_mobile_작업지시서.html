반드시 숙지 — 웹 코드와 다른 3가지
①
localStorage → AsyncStorage. @react-native-async-storage/async-storage 사용. 비동기 API라 await 필수. 같은 키 prefix "sm_" 유지. 값은 JSON.stringify/parse.
②
Notification API → expo-notifications. 브라우저 Notification 대신 expo-notifications 사용. 앱이 백그라운드 상태에서도 발송 가능. 권한 요청 코드가 다름.
③
setInterval → useFocusEffect + useRef. 앱이 백그라운드로 가면 폴링을 멈춰야 배터리를 아낌. useFocusEffect 안에서 interval 시작, cleanup에서 clearInterval.
④
Canvas → react-native-svg. Sparkline은 SVG Path로 구현. 웹 Canvas API 없음.
⑤
API URL → 환경변수. EXPO_PUBLIC_API_URL=http://[PC IP]:3000. localhost는 시뮬레이터에서만 동작, 실기기에서는 PC IP 사용.
