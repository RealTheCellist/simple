# WATCHR Platform v1.0.1

Production release for WATCHR platform expansion beyond web-only scope.

## Highlights

- Quant Lab toolkit added (`watchr-quant-lab`) with backtest, walk-forward, and data policy modes
- Shared backend API bootstrapped (`watchr-api`) with prediction service + health/realtime surfaces
- Ops strategy console added (`watchr-ops`) for runtime monitoring and control workflows
- Realtime pipeline shipped: WebSocket hub on API and live subscription in Ops with polling fallback
- Enterprise layer shipped: login auth, role-based access control, and audit trail endpoints/UI
