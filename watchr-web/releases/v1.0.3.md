# WATCHR Platform v1.0.3

Operational hardening release for production readiness.

## Included updates

- Enterprise auth hardening
  - Passwords are stored as hashes (scrypt + salt)
  - Strong password policy for user creation
  - Demo account provisioning can be disabled in production
  - Bootstrap admin credentials supported via environment variables
- Operations observability
  - Added secured endpoint: `GET /api/ops/metrics` (requires `audit:read`)
  - Request/status/auth counters exposed for runtime monitoring
- Release consistency
  - Unified package versions to `1.0.3` across web/api/ops/mobile
  - Added mobile CI workflow (`typecheck`)
  - Updated release metadata and changelog

- `v1.0.3`
