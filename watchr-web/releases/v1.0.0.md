# WATCHR Web v1.0.0

Official production release for WATCHR Web.

## Highlights

- Futures-based main-session prediction engine
- Backtest tracker with hit rate / MAE metrics
- Adaptive weight tuning mode for prediction factors
- Runtime API resilience (timeout + retry + exponential backoff)
- CI quality gates (test, build, perf budget, dist smoke)
- Production deploy pipelines (Netlify, Vercel)
- Manual rollback workflow + rollback playbook

## Operational readiness

- CI: `.github/workflows/ci.yml`
- Deploy: `.github/workflows/deploy-netlify.yml`, `.github/workflows/deploy-vercel.yml`
- Rollback: `.github/workflows/rollback-deploy.yml`
- Deployment guide: `watchr-web/DEPLOYMENT.md`
- Rollback guide: `watchr-web/ROLLBACK.md`
